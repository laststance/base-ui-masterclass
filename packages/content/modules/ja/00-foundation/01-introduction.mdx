# ヘッドレスUIライブラリとは？

**ヘッドレスUIライブラリ**は、ビジュアルスタイリングなしでUIコンポーネントの振る舞いとアクセシビリティを提供します。コンポーネントの骨格と考えてください — ロジック、キーボード操作、ARIA属性はすべて処理されますが、CSSは完全にあなたのものです。

## スタイル付き vs ヘッドレス

| 特徴 | スタイル付き (Material UI) | ヘッドレス (Base UI) |
|------|--------------------------|---------------------|
| ビジュアルデザイン | プリビルト | あなたが決める |
| CSS制御 | オーバーライド/テーマ | 完全な所有権 |
| バンドルサイズ | 大きい (CSS含む) | 最小限 |
| 柔軟性 | テーマに制限 | 無制限 |

## このコースで作るもの

このコースでは、**Base UIの全35コンポーネント**をゼロから再構築します：

- プリミティブ: Button, Input, Separator
- ステート管理: Switch, Toggle, Checkbox
- フォーム: Field, Fieldset, Radio, Select
- オーバーレイ: Dialog, Popover, Tooltip, Menu
- アドバンスド: Slider, ScrollArea, Toast
- その他...

最終的に、あなた自身のヘッドレスUIライブラリを**npmパッケージとして公開**できるようになります。

## 前提知識

- Reactの確かな知識（hooks, context, refs）
- TypeScriptの基礎（ジェネリクス、ユーティリティ型）
- HTMLセマンティクスとARIAの基本的な理解
